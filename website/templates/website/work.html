Jumping Cubes

Featuring an AI that uses minimax with alpha-beta pruning

A strategic game in which the objective is to conquer all the squares on the board. Given a n x n board with all squares initially white and spotless, red player will add a spot to a white/red square, followed by blue player adding a spot to a blue/white square. Adding a spot to a square will change the square's color to the current player if the square is white. If a square has more spots than it's neighbors, up, down, left, and right, the spots will be reallocated among its neighbors, changing the color of the neighboring squares to that of the original. This process is recursive. Once the board contains squares of only red or blue, the respective player wins.

Users can choose to play against an AI that uses a min-max algorithm with alpha-beta pruning or against other users.

Built with Java.
